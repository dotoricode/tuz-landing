# tuz-landing 프로젝트 지침

## 배포 워크플로우

**배포 전 반드시 로컬 테스트를 완료해야 합니다.**

### 로컬 테스트 체크리스트
1. 브라우저에서 `index.html` 열기
2. Pick 생성 기능 동작 확인 (큰사장/작은사장 섹션)
3. 관리자 페이지(`admin.js`) 기능 확인
4. 모바일 뷰(반응형) 확인

### 테스트 완료 후 배포
```bash
git add .
git commit -m "..."
git push origin main
```

> 훅 설정: `git push` 등 배포 명령 실행 시 자동으로 차단되며, 테스트 완료 확인 후 진행합니다.
